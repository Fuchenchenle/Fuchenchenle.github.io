知道p q e c 典型的求m

c的d次幂除以N求余数m

我们不知道d怎么办？求啊。 怎么求？套公式啊！

d = e mod phi(n)

n = p*q
phi(n) = (q-1) * (p-1)

知道了n和phi(n) 直接解出d

d = (libnum.invmod(e,fn))

知道d你套公式还套不出一个m?

完整代码

import libnum

n = p*q
fn = (p-1)*(q-1)
d = (libnum.invmod(e,fn))
flag = pow(c,d,n)
print(flag)
